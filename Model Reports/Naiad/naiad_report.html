Model Purpose

NAIAD is a specialized neural network model designed for comprehensive water system fault detection and diagnostics. As part of the Nexus BMS Neural Network Suite, it provides real-time monitoring capabilities for 16 different water system fault conditions with exceptional accuracy. The model specializes in detecting pump failures, valve malfunctions, water quality issues, and system imbalances in critical water infrastructure.

Technical Specifications

Model Architecture

  • Type: Multi-Task Neural Network for Water Systems
  • Hidden Size: 512 neurons
  • Dropout Rates: [0.1, 0.15, 0.2]
  • Parameters: 632,230 total
  • Input Features: 64 water sensors × 7 timesteps
  • Output Classes: 16 water system fault types

Training Configuration

  • Framework: PyTorch 2.6
  • Device: CUDA (GPU)
  • Batch Size: 256 (training), 128 (fine-tuning)
  • Optimizer: AdamW with OneCycleLR
  • Loss Functions: CrossEntropyLoss + Multi-task
  • Learning Rate: 0.001-0.003 (OneCycleLR)
  • Training Samples: 160,000 (perfectly balanced)
  • Epochs: 50 (balanced) + 10 (fine-tune)

Water System Sensors

  • Flow Sensors: Primary/secondary flow rates (GPM)
  • Pressure Sensors: Supply/return pressure differential
  • Temperature Sensors: Supply/return temperature monitoring
  • Pump Monitoring: Speed, power, vibration analysis
  • Valve Position: Control valve positioning feedback
  • Water Quality: Chemical composition and purity
  • System Balance: Flow distribution and efficiency

Multi-Task Learning Outputs

  • Fault Classification: 16-class water system fault detection
  • Flow Status: Low/Normal/High flow classification
  • System Balance: Balanced/Imbalanced system state
  • Pressure Prediction: Continuous pressure forecasting
  • Temperature Prediction: Heat transfer efficiency
  • Water Quality: Chemical degradation detection
