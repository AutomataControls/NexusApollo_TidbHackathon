Model Purpose

AQUILO is a state-of-the-art neural network model designed for electrical system fault detection. As part of the Nexus BMS Neural Network Suite, it provides real-time diagnostic capabilities for 13 different electrical fault conditions with exceptional accuracy. The model specializes in detecting power quality issues, phase anomalies, and electrical safety hazards.

Key Achievements

  • Perfect Detection (100%): PHASE_LOSS, PHASE_REVERSAL
  • Near-Perfect (>99%): GROUND_FAULT, VOLTAGE_SWELL
  • Production Ready: All 13 fault types exceed 90% accuracy threshold
  • V2 Improvements: Resolved zero-recall issues from V1 through enhanced training
  • Robust Performance: Balanced accuracy across all fault types

Technical Specifications

Model Architecture

  • Type: Deep Neural Network with Self-Attention
  • Hidden Size: 512 neurons
  • Dropout Rates: [0.3, 0.4, 0.5]
  • Attention Heads: 8
  • Input Features: 24 electrical sensors × 10 timesteps
  • Output Classes: 13 fault types

Training Configuration

  • Framework: PyTorch
  • Device: CUDA (GPU)
  • Batch Size: 256
  • Optimizer: AdamW with weight decay
  • Loss Function: Focal Loss with alpha=0.25, gamma=2.0
  • Learning Rate: OneCycle (max_lr=0.001)
  • Data Augmentation: Mixup (alpha=0.2)
  • Training Samples: 100,000
  • Epochs: 50

Performance Requirements

  • Inference Time: <5ms per prediction
  • Memory Usage: ~120MB model size
  • Minimum Accuracy: 85% per fault type
  • Target Accuracy: 95%+ overall
  • Safety-Critical Faults: 99%+ detection rate

Sensor Inputs

  • Voltage Measurements: 3-phase voltages (V_a, V_b, V_c)
  • Current Measurements: 3-phase currents (I_a, I_b, I_c)
  • Power Metrics: Active, reactive, apparent power per phase
  • Quality Indicators: THD, power factor, frequency
  • Ground Monitoring: Neutral current, ground fault current
  • Temporal Resolution: 10 timesteps at 100ms intervals

Integration with Nexus BMS
